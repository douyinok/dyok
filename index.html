🎬 Halsu Plugins for Streaming

OBS Studio 第三方插件全中文使用手册

Windows 64-bit OBS Studio 32.0.4+ GPL v2.0 开源 版本 1.1.1 (2026-02-09)

📋 插件套件概览

Halsu Plugins for Streaming 是一套专为 OBS Studio 设计的第三方滤镜/插件集合，主要面向绿幕抠像、透明通道处理、边缘光融合以及光学后特效等直播与视频合成场景。

🎨 HybridKeyer — 混合色度/亮度键

  • RGB + YUV 双引擎抠像
  • 去溢色 & 边缘优化
  • 支持参考帧校正不均匀光照
  • 阴影/高光精细控制

✨ AlphaTools — 透明通道工具

  • 膨胀 / 模糊 / 中值滤波
  • 去边缘污染（Unpremultiply）
  • X/Y 偏移调整
  • 灰度预览模式

💡 Relightwrap — 环境光包裹

  • 模拟背景光反射到前景
  • 方向灯光角度控制
  • 多种合成混合模式
  • 侵蚀式去彩色镶边

🔍 LensEffects — 镜头光学特效

  • 多种模糊类型 & 色差
  • 扩散辉光（Glow）效果
  • 暗角（Vignette）系统
  • 移轴/浅景深模拟
💡 硬件建议：本套件在高性能硬件上运行最佳。首次应用高强度滤镜时，建议从保守的"质量"和"半径"参数开始，逐步上调。

📦 安装指南

前置条件

项目要求
操作系统Windows 64-bit
OBS Studio 版本≥ 32.0.4（推荐最新版）
许可证GPL v2.0（开源免费）

安装步骤

  1. 下载插件包
    访问 GitHub 仓库：https://github.com/Halsu/halsu-plugins-for-streaming
    点击 Releases → 下载 Halsu_PluginsForStreaming_20260209.zip
  2. 关闭 OBS Studio
    确保 OBS 已完全退出（检查任务栏托盘）。
  3. 解压并合并文件
    将压缩包内的所有内容合并复制到 OBS 的安装目录（通常位于 C:\Program Files\obs-studio）。
    关键目录结构： 
    obs-studio/
├── data/
│   └── obs-plugins/          ← 放入插件数据文件
├── obs-plugins/
│   ├── 64bit/                ← 放入 DLL 动态库
│   └── Halsu_*.dll
└── ...
  4. 启动 OBS 验证
    打开 OBS → 右键任意视频源 → 滤镜 → 在滤镜列表中应能看到 Halsu 系列滤镜。
⚠️ 注意：如果安装后 OBS 无法启动，请检查 DLL 是否被 Windows Defender 或其他杀毒软件拦截。可将 OBS 安装目录加入白名单。

添加滤镜到来源

  1. 在"来源"面板中，右键点击目标视频源（如"捕获窗口"、"视频采集设备"等）
  2. 选择 "滤镜"
  3. 在弹出的滤镜窗口底部，点击"+"号
  4. 从列表中选择需要的 Halsu 滤镜（如 HybridKeyer、AlphaTools 等）
  5. 点击确定后即可调整参数

🎨 HybridKeyer — 混合色度/亮度键

HybridKeyer 是一个综合型抠像滤镜，同时支持 RGB 和 YUV 两种色彩空间的色度键（Chroma Key）与亮度键（Luma Key），内置去溢色和边缘精修功能。主要面向绿幕/蓝幕抠像，但也适用于其他纯色背景。

第一步：基础设置
  1. 选择背景板色调（Backdrop Hue）——即你绿幕或蓝幕的主色相。可以通过吸管工具选取。
  2. 调节白色阈值（White Threshold）和黑色阈值（Black Threshold），直到画面中的主体呈现为实心不透明，而背景完全消失变为透明。
💡 技巧：先调黑色阈值去除背景，再微调白色阈值保留主体细节。
第二步：高级抠像参数

替代键控算法（Alternative Keying Algorithm）

切换不同的抠像算法，适用于不同材质和光照条件下的背景分离。

参考帧（Reference Frame）

导入一张"干净"的背景参考图，用于校正光照不均匀导致的抠像瑕疵。

预处理去饱和 / 饱和度（Pre-Key Desaturation / Saturation）

在抠像前对色彩进行预处理，有助于更精准地识别背景色范围。

高光 & 阴影（Highlights & Shadows）

分别控制亮部和暗部的抠像强度，避免高光溢出或暗部残留。

第三步：边缘优化
  • 抗锯齿（Anti-Aliasing） — 平滑抠像边缘的锯齿感
  • 阴影着色（Shadow Coloration） — 自定义边缘阴影颜色
  • 垃圾遮罩（Garbage Matte） — 用矩形/多边形裁剪掉不需要抠像的区域
  • 内部遮罩（Inside Matte） — 从内部收缩透明区域，保护主体不被误抠
第四步：去溢色（Spill Reduction）

绿幕抠像后，人物边缘常会泛绿色。HybridKeyer 提供多种去溢色算法：

  • 标准去溢色 — 通用场景
  • 精细模式 — 保留更多发丝细节
  • 强度控制 — 调节去溢色的力度，避免过度导致肤色失真
⚠️ 常见问题：如果抠像后主体出现"空洞"，尝试降低黑色阈值；如果背景残留，尝试提高白色阈值或启用参考帧功能。

✨ AlphaTools — 透明通道工具

AlphaTools 专注于透明通道（Alpha Channel）的操控与遮罩清理。当你已经通过 HybridKeyer 或其他手段完成了初步抠像，但边缘仍有毛刺、噪点或不规则区域时，AlphaTools 是完美的后期打磨工具。

处理模式选择
模式说明适用场景
纯膨胀（Pure Expansion）将遮罩向外扩展指定像素修复抠像过紧导致的主体边缘被切掉
模糊膨胀（Blurred Expansion）先膨胀再高斯模糊需要柔和过渡的边缘
标准模糊（Standard Blur）直接对遮罩做高斯模糊柔化锯齿边缘
中值滤波（Median Filtering）去除遮罩中的小噪点孔洞抠像结果边缘有细小杂点
核心参数详解
  • 膨胀半径（Expansion Radius） — 控制遮罩向外扩展的像素量
  • 高斯软化（Gaussian Softening） — 模拟高斯模糊的强度，让边缘更自然
  • 取消预乘（Unpremultiplication） — 清除边缘的颜色污染（Premultiplied Alpha 常见问题）
  • X / Y 偏移 — 将遮罩在水平和垂直方向上做位移，用于对齐主体
  • 灰度预览（Grayscale Preview） — 以黑白灰度显示当前遮罩状态，方便精确编辑。白色=不透明，黑色=透明
推荐工作流
  1. 先用 HybridKeyer 完成基础抠像
  2. 添加 AlphaTools，开启灰度预览
  3. 选择 中值滤波 去除噪点
  4. 如有必要，切换到 模糊膨胀 柔化边缘
  5. 关闭灰度预览，观察最终效果

💡 Relightwrap — 环境光包裹

Relightwrap 模拟真实世界中背景光线反弹到前景人物边缘的物理现象。在将抠像人物合成到新背景时，如果没有环境光包裹，人物看起来就像"贴纸"一样浮在画面上。Relightwrap 能让合成效果更加逼真。

快速上手（新手推荐）
  1. 设置一个自定义的包裹色调（Wrap Hue）——通常选取背景的暖色或冷色基调
  2. 调节强度（Intensity）参数，从低往高慢慢加，直到人物边缘自然地融入背景光色
  3. 调节扩散范围（Spread），控制光包裹的宽度
高级参数

色调范围（Tonal Range）

定义哪些明暗区域会受到包裹效果影响。可以只作用于高光、阴影或中间调。

背景采样源（Background Sample Source）

  • 加载背景画面 — 从实际合成背景中采样颜色
  • 纯色采样 — 手动指定一个颜色作为环境光来源

包裹衰减（Wrap Falloff）

控制光包裹从边缘向内部的衰减速度，值越小过渡越柔和。

合成混合模式（Compositing Blend Mode）

选择包裹效果与原画面的混合方式（正常、叠加、柔光等）。

高级特性
  • 表面细节分析（Surface Detail Analysis） — 自动检测人物皮肤/衣物纹理，避免包裹效果破坏原有细节
  • 方向灯光（Directional Lighting） — 可调整光源的角度和范围，模拟不同方向的打光效果
  • 侵蚀式去镶边（Erosion-based Defringe） — 消除人物边缘因抠像产生的彩色光晕（常见于绿幕拍摄的头发边缘）
💡 最佳实践：Relightwrap 的效果高度依赖背景画面。建议先完成抠像（HybridKeyer + AlphaTools），再添加 Relightwrap 进行最后的光学融合。

🔍 LensEffects — 镜头光学特效

LensEffects 是一组光学模拟后处理滤镜，可以为你的画面添加各种摄影机镜头效果和创意视觉特效。

特效类型
特效说明
高斯模糊（Gaussian Blur）经典柔和模糊效果
径向模糊（Radial Blur）从中心向外扩散的模糊，适合速度感表现
扩散辉光（Diffusion Glow）模拟老式柔光镜效果，高光区域产生梦幻光晕
色差（Chromatic Aberration）模拟镜头色散，RGB 三通道轻微错位
暗角（Vignette）画面四角渐暗，引导视觉焦点
核心控制参数
  • 强度（Intensity） — 整体特效的强弱程度
  • 采样密度（Sampling Density） — 影响特效的细节精度和性能消耗
  • 不透明度（Opacity） — 控制特效层与原画面的混合比例
暗角系统 & 移轴模拟

LensEffects 的暗角（Vignette）功能支持两种模式：

  • 传统暗角 — 单纯让画面四角变暗，营造电影感
  • 程序化深度图（Procedural Depth Map） — 暗角区域同时调制模糊强度，实现移轴效果（Tilt-Shift）或浅景深（Soft Focus）的摄影风格
💡 移轴效果设置：将暗角中心缩小、边缘扩大，同时让暗角区域的模糊强度逐渐增加，即可模拟微距摄影中的"玩具相机"效果。

🛠️ 排错与常见问题

Q1：10-bit 画面颜色异常

症状：加载插件后，10-bit 色深的素材出现颜色偏移或偏色。

解决方案：在这些插件滤镜之前，先添加一个 OBS 自带的"颜色校正"（Color Correction）滤镜，将色彩空间转换回 8-bit，问题即可解决。

Q2：OBS 启动崩溃或闪退
  • 确认 OBS 版本 ≥ 32.0.4
  • 确认操作系统为 Windows 64-bit
  • 检查 DLL 文件是否完整复制到正确的 obs-plugins\64bit 目录
  • 暂时禁用杀毒软件/Windows Defender 实时保护
Q3：滤镜列表中看不到 Halsu 插件
  1. 确认 OBS 已完全关闭后重新安装（不是覆盖，而是先删除旧文件再复制）
  2. 检查 obs-plugins\64bit 目录下是否有 Halsu_*.dll 文件
  3. 检查 data\obs-plugins 目录是否存在且包含资源文件
  4. 以管理员身份运行 OBS
Q4：性能下降 / 卡顿
  • 降低滤镜的"质量"参数
  • 减小"半径"（Radius）值
  • 减少同时使用的滤镜数量
  • 考虑升级 GPU（这些插件主要依赖 GPU 加速）
Q5：边缘有彩色光晕（Color Fringing）

这是绿幕抠像的经典问题，按以下顺序处理：

  1. 在 HybridKeyer 中增强去溢色（Spill Reduction）
  2. 在 AlphaTools 中使用中值滤波清理边缘噪点
  3. 在 Relightwrap 中启用侵蚀式去镶边（Defringe）
